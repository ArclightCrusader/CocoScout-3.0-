# CocoScout-3.0-
Scouting Web-App for the 2018 season

The new scouting app will include:
+ BlueAlliance & Spyder data
+ An algorithm for the best pick
+ Data displays in form the of graphs, charts, etc

+ GitHub Pages
+ [Firebase][2]
+ [Cloudflare][3]
+ [Docker][4]
